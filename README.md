# Email-Spam-Detector
##  Description
This project detects whether an email is spam or not using machine learning.

##  Technologies
- Python
- Scikit-learn
- Streamlit

##  How to Run
1. Install requirements
2. Run train.py
3. Run streamlit app

## Features
- Spam detection
- Probability score
- Visualization
